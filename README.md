V2.0
新功能
2025-04-15
▸
新功能線芯自動排列線色：輸入線芯數量後，各芯序依循環順序（棕→紅→橙→黃→綠→藍→紫→灰→白→黑）自動填入線色預設值，已手動填寫的顏色不受影響

▸
新功能載入備份繼續編輯：新增「📂 載入 Excel」按鈕，可載入由本系統匯出的 .xlsx 檔案，完整還原所有線材資料（Handler Model、線名、線長、線徑、各芯序線色/線碼/端子/備註）繼續編輯

▸
改善載入錯誤時顯示詳細步驟紀錄面板，方便診斷問題


V1.x
初始版本
2025-04-01

▸
線材組態輸入：Handler Model、線名、線長、線徑（AWG / mm²）、線芯數量

▸
線組資訊：各芯序線色（含圓點預覽）、線碼、左右端端子兩層下拉、芯序備註

▸
內建端子型錄：六大類共 54+ 種規格（歐式、母壓排、R型、Y型、全絕緣母端子、AMP PIN針）

▸
歐式端子下拉以「顏色（型號）（AWG/mm²）」格式顯示

▸
母壓排智能鎖定：依線芯數量自動決定 P 數，任一芯序選取後同步全部芯序

▸
線材總表：依 Handler Model 分組，支援編輯與刪除

▸
匯出 Excel：彩色格式，線色欄套用對應背景色，左右端類型標籤色，多芯合併儲存格，機器分頁

▸
匯出規格欄位完整帶入選單顯示內容（含 AWG / mm² 資訊）

▸
相同端子類型/規格時跨芯序合併儲存格

▸
匯出檔名格式：{機型}_線材總表_{日期}.xlsx

▸
線組備註（整組）與芯序備註（各芯獨立）
